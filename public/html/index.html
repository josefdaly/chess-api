Joe's Chess API

Welcome. This is a JSON API for chess. Each endpoint takes a string of moves in in the path. Example: http://www.chess-api.herokuapp.com/next_best/a2a3e7e5

Documentation

/next_best/:moves

Returns an AI next move in the response JSON.

/valid_move/:moves

Determines the legality of the last move. Assumes legal moves up to that point.

/status/:moves

Returns the status of the game. Possible values:

  • in_progress
  • white_won
  • black_won
  • white_won_resign
  • black_won_resign
  • stalemate
  • insufficient_material
  • fifty_rule_move
  • threefold_repitition
  • unknown

/board_string/:moves

Returns a string representation of the board.
